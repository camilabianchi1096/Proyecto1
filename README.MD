# Proyecto de pruebas automatizadas utilizando Selenium WebDriver, implementando el patrón Page Object Model (POM) y gestionando suites con TestNG.

## Características principales

✅ Lenguaje: Java
✅ Framework de pruebas: TestNG
✅ Patrón de diseño: Page Object Model (POM)
✅ Gestión de suites: Configuración mediante testng.xml
✅ Automatización orientada a escalabilidad y mantenimiento

```bash
src/test/java
 ├── pages        # Clases POM 
 ├── utils        # Utilidades y helpers
 ├── test         # Casos de prueba con TestNG
 └── resources              
 		 └── testngparalelo.xml # Suite de test NG
 		 └── testngsmoke.xml 	# Suite de test NG
```
